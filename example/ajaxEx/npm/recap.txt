npm

node package manager

api나 라이브러리를 사용하기 위해서는 다운받아야 한다.
그리고 그것을 다운받는 저장소는 여러 종류가 있는데
maven, apt, pip 등등이 있고
npm은 그중의 하나다.

그리고 이 저장소를 관리해준다.
spring에 maven이 있다면 nodejs 에는 npm이 있다.

즉 필요한 라이브러리를 아주 쉽게 다운받고 적용할 수 있다.

npm 사이트에 가서 원하는 소프트웨어를 검색하고, cli에 입력하면 된다.
만약 특정한 재료로 쓰는게 아니라면(특정 디렉토리 에서만)
전역 명령어(-g)를 입력함으로서 모든 디렉토리에서 해당 프로그램을
사용할 수 있게 할 수 있다.
